## 함수
함수란 하나의 로직을 재실행 할 수 있도록 하는 것으로 코드의 재사용성을 높여줌

{% highlight js %}
function 함수명( [인자...[,인자]] ){
    코드 내용
       return 반환값
}
{% endhighlight %}


## return
함수를 종료하고 return 뒤의 값을 반환함.

## argument
함수에 값을 전달할때 쓰는 걸 인자라고 함.
